# 📝 To-Do List App

A simple and responsive To-Do List application built with HTML, CSS, and JavaScript. Users can add, complete, and delete tasks, and data is saved using localStorage.

## Features
- ➕ Add Tasks
- ✅ Mark Tasks as Complete
- 🗑️ Delete Tasks
- 💾 Save Tasks using localStorage
- 📱 Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
